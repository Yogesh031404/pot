üìã Setup Instructions

1. Create Google Spreadsheet

  1. Go to Google Sheets
  2. Create a new spreadsheet named "EcoPots_Student_Registrations"
  3. Copy the spreadsheet ID from the URL (the part between /d/ and /edit)

2. Set up Google Apps Script

  1. Go to Google Apps Script
  2. Create a new project
  3. Copy the entire contents of Google_Apps_Script_Code.gs
  4. Replace YOUR_SPREADSHEET_ID_HERE with your actual spreadsheet ID
  5. Save the project (Ctrl+S)

3. Deploy as Web App

  1. Click "Deploy" ‚Üí "New deployment"
  2. Click "Select type" ‚Üí "Web app"
  3. Set "Execute as" to "Me"
  4. Set "Who has access" to "Anyone"
  5. Click "Deploy"
  6. Copy the Web app URL

4. Test Integration

  1. Paste the Web app URL and Spreadsheet ID above
  2. Click "Save Configuration"
  3. Click "Test Connection"
  4. If successful, try "Test Sample Submission"
